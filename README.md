# PartyChat

Вёрстка и сборка чата, 1 спринт

## Команды

  `npm run build` - сборка страниц из шаблонов папки src/pages
  
  `npm run start` - сборка страниц из шаблонов папки src/pages и запуск epress-сервера на 3000 порту
